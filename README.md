# Snake Grid Portfolio 🐍✨

An interactive, game-inspired portfolio for Mandar Shimpi, an AI/ML developer from COEP Pune. This project features a custom-built Snake game dashboard that doubles as a navigation system.

## 🚀 Features

- **Interactive Snake Dashboard**: Navigate through the portfolio by playing the classic Snake game. Eat "portals" to enter different sections.
- **Triple-Mode Theme System**: Seamlessly switch between **Light**, **Dark**, and **System** themes. Your preference is remembered across sessions.
- **Bento Grid Projects**: A modern, clean presentation of projects like **Resume Screener Pro**.
- **Fully Responsive**: Optimized for both desktop and mobile devices with touch-swipe controls.
- **Email Integration**: Functional contact form connected to Formspree.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern styling and responsive design.
- **JavaScript (Vanilla)**: Game logic, theme engine, and progress tracking.
- **CSS Variables**: Dynamic color architecture for instant theme switching.

## 📁 Directory Structure

- `index.html`: Dashboard & Snake Game.
- `about.html`: Professional profile & academic details.
- `projects.html`: Work showcase.
- `skills.html`: Technical expertise breakdown.
- `contact.html`: Form & social nodes.
- `portfolio_demo.webp`: Demonstration video.

## 📖 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/MandarShimpi/SnakeGridPortfolio.git
   ```
2. Open `index.html` in any modern web browser.
3. (Optional) For a local dev environment, run a live server (e.g., `npx serve .`).
